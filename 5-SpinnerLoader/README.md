# 🔄 Spinner Loader

A collection of modern animated CSS spinner loaders built using pure HTML and CSS.

This project showcases several loading spinner styles commonly used in modern web applications. Each loader demonstrates a different animation technique and visual design while requiring **no JavaScript**.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- Multiple spinner styles
- Pure HTML & CSS
- Smooth CSS animations
- Modern dark UI
- Responsive layout
- Easy to customize
- No JavaScript required

---

## 🛠 Technologies

- HTML5
- CSS3
