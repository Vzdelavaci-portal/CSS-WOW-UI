# 🌊 Liquid Button

A modern animated button with realistic liquid effects built using HTML, CSS and JavaScript.

This project demonstrates how to create interactive liquid-style buttons featuring animated waves, floating bubbles, glowing colors and smooth hover effects.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- Animated liquid effect
- Realistic wave animation
- Floating bubbles
- Animated light reflection
- Smooth hover interaction
- Click animation
- Multiple color themes
- Responsive design
- Pure HTML, CSS and JavaScript

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
