# 💎 Glass Card

A premium glassmorphism profile card built with HTML, CSS and JavaScript.

This project demonstrates a modern glass-style UI featuring animated lighting, 3D hover effects, floating elements and interactive components inspired by premium web interfaces.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- Glassmorphism design
- 3D hover interaction
- Mouse-follow lighting effect
- Animated background blobs
- Floating avatar animation
- Interactive Follow button
- Responsive layout
- Pure HTML, CSS and JavaScript
