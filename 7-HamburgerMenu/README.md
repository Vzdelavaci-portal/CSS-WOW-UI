# 🍔 Hamburger Menu

A modern animated hamburger menu built with HTML, CSS and JavaScript.

This project demonstrates a responsive fullscreen navigation menu featuring a smooth hamburger-to-close animation, glassmorphism effects and elegant transitions.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- Animated hamburger icon
- Smooth menu opening animation
- Fullscreen navigation
- Glassmorphism background
- Backdrop blur effect
- Hover animations
- Responsive design
- Pure HTML, CSS and JavaScript

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
