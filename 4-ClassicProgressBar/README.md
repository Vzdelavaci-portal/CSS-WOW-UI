# 📊 Classic Progress Bar

A clean and modern animated progress bar built with HTML, CSS and JavaScript.

This project demonstrates how to create a responsive progress bar with smooth animations, percentage tracking and interactive controls.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- Animated progress bar
- Live percentage display
- Start and Reset controls
- Smooth filling animation
- Modern dark UI
- Gradient progress effect
- Responsive design
- Pure HTML, CSS and JavaScript

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
