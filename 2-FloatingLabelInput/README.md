# ✨ Floating Label Input

A modern animated input form built with HTML, CSS and JavaScript.

This project demonstrates floating labels, smooth focus effects, glowing inputs and a clean modern user interface.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- Floating label animation
- Modern dark UI
- Smooth focus glow effect
- Animated icons
- Responsive design
- Validation-ready inputs
- Pure HTML, CSS and JavaScript

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
