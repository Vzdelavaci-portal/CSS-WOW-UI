# 🧊 Glassmorphism Button

A modern glass-style button built with HTML, CSS and JavaScript.

This project showcases the popular Glassmorphism design trend, combining transparency, blur effects, animated reflections and smooth interactions.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- Glassmorphism design
- Blur and transparency effects
- Animated light reflection
- Smooth hover animation
- Click animation
- Animated background blobs
- Responsive design
- Pure HTML, CSS and JavaScript

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
