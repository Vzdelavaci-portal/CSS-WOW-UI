# 🍏 iOS Toggle Switch

A modern iOS-inspired toggle switch collection built with HTML, CSS and JavaScript.

This project demonstrates elegant toggle switches with smooth animations, realistic shadows and responsive interactions inspired by Apple's Settings application.

Part of the **CSS WOW UI** collection.

---

## ✨ Features

- iOS-inspired toggle switches
- Smooth toggle animations
- Realistic shadows
- Modern glass UI
- Responsive design
- Interactive click animations
- Multiple settings examples
- Pure HTML, CSS and JavaScript

---

## 🛠 Technologies

- HTML5
- CSS3
- JavaScript
